To update a provider to its latest version, you can use:

```sh
devsy provider set-source <name>
```

To update to a specific version, pass `--version <tag>`:

```sh
devsy provider set-source <name> --version v0.2.0
```

To list the available versions for an installed provider, use:

```sh
devsy provider versions <name>
```

## From GitHub

Similar to adding a provider, you can update via:
```sh
devsy provider set-source <provider-name> my-org/my-repo
```

If you want to update (or downgrade) a specific version, you can do:
```sh
devsy provider set-source <provider-name> github.com/my-org/my-repo@v0.2.0
```

## From URL

You can also specify the URL to the `provider.yaml` file, for example:

```sh
devsy provider set-source <provider-name> http://path/to/my/provider.yaml
```

### From File

Run the following command to update provider `provider-name`:
```sh
devsy provider set-source <provider-name> ../path-to-updated/provider.yaml
```
