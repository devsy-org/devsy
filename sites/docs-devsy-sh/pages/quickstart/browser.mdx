## Quickstart with VS Code Browser

Make sure you have installed [Devsy](../getting-started/install.mdx) on your system.

### Add a Provider

<AddProvider />

### Start a Workspace with VS Code Browser

Navigate to 'Workspaces' > '+ Create' to launch the workspace wizard. Step through:

1. **Provider** – pick an initialized provider.
2. **Source** – enter your project URL or choose one of the quickstart templates.
3. **IDE** – select **VS Code Browser**.
4. **Review** – confirm the configuration.
5. **Launch** – start the workspace.

<figure>
  <img src="/docs/media/create-workspace-vscode-browser.png" alt="Devsy wizard – select VS Code Browser" />
  <figcaption>Select 'VS Code Browser' as the IDE</figcaption>
</figure>

The wizard streams logs as Devsy starts the workspace. Once it's ready, VS Code opens in your browser connected to the DevContainer.
