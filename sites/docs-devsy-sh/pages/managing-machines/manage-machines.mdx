## Create a machine

Create a new machine with:

```sh
devsy machine create <name-of-machine> --provider <provider-name>
```

List all machines:

```sh
devsy machine list
```

Example output:

```
  NAME | PROVIDER | AGE
-------+----------+------
  <name-of-machine> | aws      | 21s
```

Check the state of a machine:

```sh
devsy machine status <name-of-machine>
```

Example output:

```
08:48:58 info Machine '<name-of-machine>' is 'Running'
```

## SSH into a machine

Open an SSH session directly to the machine:

```sh
devsy machine ssh <name-of-machine>
```

## Stop a machine

Stop a machine with:

```sh
devsy machine stop <name-of-machine>
```

Check the status afterwards:

```sh
devsy machine status <name-of-machine>
```

Example output:

```
08:58:58 info Machine '<name-of-machine>' is 'Stopped, you can start it via 'devsy machine start <name-of-machine>''
```

## Delete a machine

Delete a machine with:

```sh
devsy machine delete <name-of-machine>
```

:::warning
This is non-reversible. All workspace containers and data on the machine are lost.
:::
